# Progress System Core – ESR & Level (Summary)

- ESR: skill rating, 0–5000, used for matchmaking and ranks.
- Ranks: Beginner, Rookie, Pro, Ace, Legend (each with 3 divisions).
- Level: separate from ESR, uses XP from matches & platform activity.
- Visual:
  - Level = ring around avatar
  - ESR = bar + rank icon
