# Project Rules

1. All code must be in TypeScript.
2. Do not write insecure code.
3. All DB access goes through Prisma or SQL wrappers.
4. Steam Auth is the ONLY login method.
5. Every match server MUST auto-shutdown.
6. Anticheat endpoints MUST log everything.
7. No unapproved libraries.
8. All changes must follow the neon-green brand.
9. No hardcoded secrets — use environment variables.
10. All backend services must be compatible with Cloud Run.
