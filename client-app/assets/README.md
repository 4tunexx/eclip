# Icons

Place your application icon here as `icon.ico`.

## Requirements

- Format: `.ico` (Windows Icon)
- Recommended sizes: 256x256, 128x128, 64x64, 48x48, 32x32, 16x16
- Use transparent background

## Creating an Icon

### Online Tools
- [ICO Convert](https://www.icoconverter.com/)
- [Favicon.io](https://favicon.io/favicon-converter/)

### From PNG
1. Create a 256x256 PNG with your logo
2. Upload to an ICO converter
3. Download as `icon.ico`
4. Place in this folder

## Temporary Solution

For testing, you can use any .ico file or generate one from a PNG. The app will work without an icon, but Windows will use a default icon.
