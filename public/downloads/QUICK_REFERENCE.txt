```javascript

function launchAnticheat() {
  const params = new URLSearchParams({
    userId: currentUser.id,
    matchId: currentMatch.id,
    token: session.token
  });
  
  // Try protocol first (installed users)
  setTimeout(() => {
    window.location.href = 'eclip://launch?' + params;
  }, 0);
  
  // Fallback to download after 2 seconds
  setTimeout(() => {
    window.location.href = '/api/download/client';
  }, 2000);
}
```

## Troubleshooting

**Q: Download fails?**  
A: Check `/api/download/client` endpoint is working. Verify file at `public/downloads/EclipAC-Setup.exe`.

**Q: App won't launch from browser?**  
A: Run `install.bat` as administrator to register protocol handler.

**Q: High CPU usage?**  
A: Normal during process scans. Background monitoring optimized.

**Q: Anticheat crashes?**  
A: Ensure Windows 10+, try reinstalling via `install.bat`.
